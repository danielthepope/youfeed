YouFeed RSS is not dead yet!

YouFeed takes a YouTube channel or playlist link and turns it into a feedly page that you can subscribe to.

Have you tried the YouFeed bookmarklet?

Drag this to the bookmarks bar for one-click YouTube to feedly conversion.

Open in feedly

Paste in the YouTube channel/playlist page URL

Insert the URL for a YouTube channel or playlist, and the corresponding feedly page will appear in a new tab.

For more info, click Help at the top of this page.

This website does not use cookies or server-side scripts. Find out more.

If a URL you submit doesn't work as expected, or if you just want to say hi, please get in touch.

YouFeed is an open source project by @danielthepope that makes it easier to subscibe to feeds using feedly.

Privacy

YouFeed does not collect any data about you or your usage at all.

You're welcome.

All data processing takes place solely on your computer, and no cookies of any kind are stored or eaten.

However, this does mean that if anything goes wrong, I don't receive any automatic notification. Therefore, if you attempt to open a link that you think should work but doesn't, please get in touch making sure you include the broken link.
